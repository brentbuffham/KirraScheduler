## Architecture Decisions

1. **Lazy initialisation** — Three.js scene only created when 3D PLAYBACK tab is first clicked. No WebGL overhead on other tabs.
2. **Local origin transform** — Large UTM coordinates handled by subtracting centroid from all positions (matching 2D canvas approach per workspace rules).
3. **Z-up convention** — Camera.up set to (0,0,1) to match mining coordinate system (Z = elevation).
4. **Pan-default mouse buttons** — Left=Pan, Middle=Dolly, Right=Rotate per workspace rules.
5. **No template literals** — All string concatenation uses `"..." + variable + "..."` per project rules.
